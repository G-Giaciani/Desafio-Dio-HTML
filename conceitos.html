Estrutura Básica

(Voltar para o índice)

HTML, abreviação de Hypertext Markup Language, é a linguagem padrão para a criação de páginas web. A estrutura básica de um documento HTML inclui tags como <html>, <head> e <body>. <html> define o início e o fim do documento HTML. <head> contém informações sobre o documento, como o título da página e metadados. <body> contém o conteúdo principal da página, como texto, imagens, vídeos e outros elementos.

Tags

Tags são os blocos de construção fundamentais de uma página HTML. Cada tag é identificada por seu nome entre colchetes angulares, como <tag>. Existem diferentes tipos de tags para diferentes finalidades, como <p> para parágrafos, <h1> a <h6> para títulos e <ul> e <ol> para listas.

Atributos de tags

Os atributos das tags HTML fornecem informações adicionais sobre os elementos. Eles são definidos dentro das tags com um nome e um valor, separados por um sinal de igual (=) e aspas duplas. Esses atributos podem controlar a aparência, comportamento e funcionalidade dos elementos. Existem atributos globais, que podem ser usados em qualquer elemento HTML, e atributos específicos de cada tipo de elemento.

Textos.

O texto em HTML é normalmente envolto pela tag <p> para parágrafos ou pode ser inserido diretamente dentro de outras tags, como <strong> por exemplo.

Listas

HTML suporta dois tipos principais de listas: listas ordenadas e listas não ordenadas. Em listas ordenadas, os itens são numerados ou marcados com um símbolo específico, como números ou letras, como no exemplo abaixo.

  1. Item 1
  2. Item 2
Em listas não ordenadas, os itens são marcados com marcadores, como pontos ou círculos.
  • Item 1
  • Item 2

Links

(Voltar para o índice)

Os links em HTML são criados com a tag <a>, que significa âncora. O atributo href dentro da tag <a> especifica o URL para o qual o link aponta. Texto ou outros elementos dentro da tag <a> são clicáveis e levam o usuário para o URL especificado, podendo ser um redirecionamento dentro da própria página ou para um site.
